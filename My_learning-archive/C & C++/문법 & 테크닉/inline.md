### ✅ `inline`의 의미


```
inline int add(int a, int b) 
{     
return a + b; 
}
```

이렇게 선언하면, 컴파일러는 이 함수를 호출할 때 
**함수 호출을 하지 않고, 해당 코드 자체를 복사해서 삽입**하려고 해.


### ✅ 주의사항

- 함수가 너무 길면 `inline` 효과가 없거나 무시됨 (컴파일러가 알아서 판단)
    
- 재귀 함수는 `inline` 안 됨
    
- 코드 크기 커질 수 있음 (함수 본문을 계속 복사하니까)

### ✅ 언제 쓰면 좋을까?

- 아주 짧고 자주 호출되는 함수
    
- 예: `color()`, `min()`, `max()` 같은 헬퍼 함수들
