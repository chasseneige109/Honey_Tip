| 함수          | 설명                         |
| ----------- | -------------------------- |
| `s.push(x)` | 스택에 요소 `x`를 **추가**         |
| `s.pop()`   | **맨 위 요소 제거** (반환 X)       |
| `s.top()`   | **맨 위 요소 반환** (제거 X)       |
| `s.empty()` | 비었는지 확인 (`true` / `false`) |
| `s.size()`  | 스택에 들어있는 요소 개수 반환          |
