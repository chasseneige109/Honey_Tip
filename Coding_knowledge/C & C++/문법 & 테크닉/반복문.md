

## ✅ 범위 기반 for 문 
```
string s = "abc";
for (char c : s) 
{
    cout << c << '\n';
}

```

> 문자열 `s`의 **모든 문자(char)** 를 하나씩 **왼쪽 변수 `c`에 넣어서** 반복해라

## ✅ `break` , `continue`

| 키워드        | 의미                       |
| ---------- | ------------------------ |
| `break`    | **반복문 전체 종료**            |
| `continue` | **현재 반복만 건너뛰고 다음 반복 계속** |
